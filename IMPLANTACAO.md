# Guia de Implantação e Infraestrutura

## Pré-requisitos de Ambiente (Local)

Para rodar este projeto no seu computador você precisa ter instalado:

- [ ] PHP (Versão 8.2 ou superior);
- [ ] Composer (Gerenciador de pacotes do PHP)
- [ ] Filament (Framework para laravel)
- [ ] NPM
- [ ] Banco de dados MySQL

## Checklist de Instalação

- [ ] Criar a estrutura da base de Dados no Laravel
- [ ] Instalar o Filament
- [ ] Configurar o arquivos de variáveis de ambiente ('.env').
- [ ] Rodar as Aplicações do banco de dados ('migrations').
