# ***Requisitos funcionais e não funcionais***

## 1.Requisitos Funcionais

- [RF01]Cadastrar usuário
- [RF02]Cadastrar o produto
- [RF03]Adicionar o produto
- [RF04]Dar baixa no sistema
  
## 2.Requisito NÃO Funcional

- [RNF01]dados dos usuários criptografados
- [RNF02]os sistema deve ser compativel com o Chrome e firefox
- [RNF03]o sistema deve realizar backups automáticos

## 3. Regra de Negócio

- [RN01]Para dar baixa no sistema:
  - tem que utilizar o que tem no sistema menos o valor informado pelo usuário
- [RN02]Para adicinar no sistema:
  - o sistema deve utilizar o valor informado e  somar com o presente no sistema

- [RN03]Limite de  baixa
  - O sistema precisa de pelo menos 1 produto dentro do sistema
